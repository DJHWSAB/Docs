# mac

## 1. mac下查看文件路径

  1. 打开终端，在 `访达` 中把文件拉进终端

  2. 在 `访达` 中选中文件，⌘ + ⌥ + c，就复制到剪贴板了


  3. **Esc** 退出编辑模式

  4. 按 **:wq** 键 保存退出vim模式

## 2. 常用的快捷键

  1.显示/隐藏以.开头的隐藏文件

  ```sh
  shift + ⌘ + .
  ```

  2. 终端获取当前目录下的所有文件包括以.开头的隐藏文件

  ```sh
  ls -a
  ```

  3. 删除文件夹
  ```sh
  rm -rf xxx
  ```

  4. 关闭程序
  ```sh
  ⌘ + q
  ```

  5. 关闭当前标签:
  ```sh
  ⌘ + w
  ```

  6. 新建标签页: 
  ```sh
  ⌘ + t
  ```

  7. 新建窗口: 
  ```sh
  ⌘ + n
  ```

  8. 调整窗口: 三指

  9. 清除当前内容: 
  ```sh
  ⌘ + u / ⌘ + 清除键
  ```

  10. 进入xx目录: 
  ```sh
  cd xx
  ```
  
  11. 退回上一级目录: 
  ```sh
  cd ..
  ```

  12. 系统根目录: ~

  13. 桌面: desktop

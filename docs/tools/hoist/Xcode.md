# Xcode

## 1. 安装Xcode

> 直接去App store下载即可

  ![xcode](/xcode_01.png)


## 2. 配置Xcode

  > 在 `mac`终端运行命令的时候会被提示没有同意 `xcode`证书 ，这个时候需要在 `Terminal`中同意 `license`。
  
  1. 打开终端输入 `sudo xcodebuild -license`，会看到一大堆的 `license`说明，直接按下空格浏览。
  
  ![xcode](/xcode_02.png)

  2. 根据提示输入 `agree` ,回车就搞定了

  ![xcode](/xcode_03.png)

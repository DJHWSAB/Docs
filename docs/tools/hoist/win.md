# win

## 常见的快捷键

  ```sh
  alt + F4  //关闭程序
  Ctrl + w  // 关闭当前标签
  Ctrl + t  // 新建标签页
  Ctrl + n  // 新建窗口
  win + 方向键 // 调整窗口: 
  Ctrl + u  // 清除当前内容
  cd xx // 进入xx目录
  cd .. // 退回上一级目录: 
  ~ // 系统根目录: 
  desktop // 桌面: 
  rm -rf xxx  // 删除文件夹
  F2  // 重命名: 
  win + d // 显示/隐藏程序: 
  start xx  // 打开xx文件:

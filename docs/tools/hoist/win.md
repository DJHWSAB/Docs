# win

## 1. 常见的快捷键

  1. 关闭程序: 
  ```sh
  alt + F4
  ```

  2. 关闭当前标签: 
  ```sh
  Ctrl + w
  ```

  3. 新建标签页: 
  ```sh
  Ctrl + t
  ```

  4. 新建窗口: 
  ```sh
  Ctrl + n
  ```

  5. 调整窗口: 
  ```sh
  win + 方向键
  ```

  6. 清除当前内容: 
  ```sh
  Ctrl + u
  ```

  7. 进入xx目录: 
  ```sh
  cd xx
  ```

  8. 退回上一级目录: 
  ```sh
  cd ..
  ```

  9. 系统根目录: ~

  10. 桌面: desktop

  11. 删除文件夹
  ```sh
  rm -rf xxx
  ```
  
  12. 重命名: F2

  13. 显示/隐藏程序: win + d

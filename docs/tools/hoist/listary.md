# Listary

  > 可简单理解为是搜索本地文件入口，使用上有点类似搜索引擎,如果你是win用户,那么这款软件适合你,mac用户请回避!!!

  1. [下载地址](https://www.listary.com/download)
  
  > 安装方式默认是傻瓜式安装,只需要更改保存地址,其他只需默认即可

  2. 文件搜索: keyword:xxx

  3. 快速启动 git Bash

  ![listary](/listary_01.png)

  4. 显示/隐藏 listary: 按下 Ctrl 两次

  5. 使用 Vscode 编辑: alt + shift + c

  6. 预览图片: alt + p
